Adds customizer settings and controls to the Gogo Theme.

== Description ==

Hunk Companion contain all features which are required to create a complete website. Main motive behind this plugin is to boost up functionality of ThemeHunk themes. This plugin will work for Gogo WordPress theme and upcoming ThemeHunk themes.

**Key Features:-**

**Features:**
- Add service section
- Add Team Section
- Add clients-and-testimonials section
- Add about-us section 
- Add slider-typewriter section
- Add call-to section
- Add woocommerce section
- Add portfolio section
- Add pricing section
- Add video-ribbon section
- Add blog section
- Add contact-us section
- Add social section
- Add Homepage Layout section

== Installation ==

1. Upload the `hunk-companion.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
